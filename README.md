# Wish list

Demo wish list app with Vue, Vuex and TypeScript.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run unit tests
```
yarn test:unit
```

### Run end-to-end tests
```
yarn test:e2e
```

### Lints and fixes files
```
yarn lint
```
